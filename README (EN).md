# Vynora 🎶

![Vynora Icon](Vynora.png)

Vynora is a premium music player built with Electron, inspired by the classic iTunes CoverFlow aesthetic, featuring modern fluid animations and glassmorphism design.

## 🚀 Features

- **Liquid Mercury Visualizer**: Dynamic volume animations with mercury fluid effects.
- **iTunes API Integration**: Automatic high-quality cover art fetching.
- **Smart Cache**: Covers are extracted from local metadata or cached on disk for instant loading.
- **Multi-platform**: Native support for Linux (AppImage, deb, pacman), Windows, and macOS.
- **Premium Design**: Responsive interface, dark mode, and smooth micro-interactions powered by Framer Motion.

## 🛠️ How to Install (Arch Linux)

On Arch Linux, you can build and install directly using our PKGBUILD:

```bash
makepkg -si
```

For other platforms, check the `dist-electron` folder after running the build.

## 💡 About

This project started as an experiment based on a Figma design and evolved into a functional, highly optimized player. Created for fun to explore complex animations in Electron.
